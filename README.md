# LeetCode Solutions 🚀

A collection of my solutions to [LeetCode](https://leetcode.com/) problems, organized by difficulty and topic. This repo is a personal log of my problem-solving journey — tracking progress, patterns, and the occasional "why didn't I see that sooner" moment.

## 📌 About

This repository contains my solved LeetCode problems, **automatically synced from LeetCode using a browser extension** ([SyncLeetCode](https://github.com/LeetSync/LeetSync)) every time a submission is accepted. Each push includes the solution code and, depending on the extension's settings, problem details like the prompt, difficulty, and topics.

> ⚡ Solutions here are added automatically — I don't manually upload them. Some folder/file naming may follow the extension's default conventions rather than the structure below exactly.
